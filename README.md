pyddp
=====

A Python 2 distributed data protocol (DDP) client.

_Warning_: This library is still in the planning stage. If you choose to use it,
           please use a specific commit.


Installation
------------

Install via `pip`

```Shell
    $ pip install pyddp
```


Get started
-----------

```Python
import ddp
```
